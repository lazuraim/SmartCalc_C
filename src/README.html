SmartCalc

Расширенная версия калькулятора с возможностью построения графиков

Поддерживаются следующие операции

  • Сложение: +
  • Вычитание: -
  • Умножение *
  • Деление: /
  • Возведение в степень: ^
  • Остаток от деления: mod

    • Функции

  • Синус - sin(x)
  • Косинус - cos(x)
  • Тангенс - tan(x)
  • Арксинус - asin(x)
  • Арккосинус - acos(x)
  • Арктангенс - atan(x)
  • Квадратный корень - sqrt(x)
  • Натуральный логарифм - ln(x)
  • Десятичный логарифм - log(x)
